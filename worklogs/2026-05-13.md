## Talos Kubernetes Cluster Bootstrap

### 手動作業（Non-IaC）

- Tailscale subnet route の設定（Talos ノードと同じ LAN 上の VM で実行）
  - `sudo tailscale set --advertise-routes=10.240.0.0/16`（Talos 初回 boot は DHCP のためサブネット全体）
  - IP forwarding の有効化（`/etc/sysctl.d/99-tailscale.conf`）
  - Tailscale admin console で subnet route を approve

- Terraform モジュールに `started` 変数を追加

- Talos VM の ISO/boot order を変数化

  - `talos/Taskfile.yml` - talos operations
  - `.sops.yaml` - SOPS encryption rules
- devcontainer に talhelper, sops, age を追加

- `.gitignore` に `.local/`, `talos/clusterconfig/` を追加
- ドキュメント作成・更新
