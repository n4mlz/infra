- `controllers/kustomization.yaml` に 4 コンポーネントを追加

- `config/kustomization.yaml` に storage/minio を追加

## 判断

- DirectPV / MinIO Operator は vendor manifest の namespace を尊重: directpv / minio-operator
- remote kustomize を Flux が直接参照する形を避け、manifest を Git に固定
- MinIO の root credential は 1Password (ClusterSecretStore: onepassword-infra) から ESO 経由で取得
- Terraform の disk 変数名は実装 (Longhorn/MinIO) ではなく用途 (persistent_volume / object_storage) で命名
- ディレクトリ名・namespace 名から `storage-*` prefix を除去 (vendor の default namespace と統一)
- vendored resource には README.md で生成元と更新手順を明記 (gateway-api-crds のパターンに準拠)
